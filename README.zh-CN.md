
#### 安装依赖

```shell
pnpm install
```

#### 启动开发环境

```shell
pnpm dev
```

#### 构建打包

```shell
pnpm build
```

#### 预览

```shell
pnpm preview
```

#### 代码检查

```shell
pnpm lint
```

#### TS 类型检查

```shell
pnpm type-check
```

#### 代码格式化

```shell
pnpm format
```

#### 运行单元测试

```shell
pnpm test
```

#### 运行单元测试并生成测试报告

```shell
pnpm test:coverage
```
